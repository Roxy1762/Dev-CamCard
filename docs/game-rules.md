# 课表风暴：规则冻结（MVP）

## 基础参数

- 生命值：32
- 起始套牌：12
- 手牌上限：5
- 先手首回合抽 4
- 后手首回合抽 5
- 三栏商店：每栏 2 张
- 固定补给：3 个无限牌堆
- 日程槽：每人 2
- 预约位：每人 1

## 核心机制

- 资源：购买卡牌
- 攻击：用于攻击玩家/场馆
- 防备：跨至对手回合，优先抵挡玩家伤害
- 压力：不可打出，回合结束弃置
- 场馆：留场可启动
- 值守场馆：存在时必须优先被攻击

## 卡牌规则类型

- action
- venue（含子类型 guard）

其他题材标签仅作 tags，不作为规则类型。

## 关键流程

### 回合开始
1. 清空自己残留防备
2. 处理延迟效果
3. 重置场馆启动次数
4. 结算日程槽安排牌

### 主要阶段（任意顺序）
- 打出行动牌
- 启动场馆
- 预约（每回合最多 1 次）
- 购买公开牌 / 预约牌 / 固定补给
- 分配攻击

### 回合结束
1. 弃本回合打出的行动牌
2. 弃手牌
3. 弃压力
4. 场馆留场
5. 抽到 5 张
